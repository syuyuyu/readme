### 本地安裝

1.確認本機環境已下載 npm 及 Node.js

2.clone 專案至本機

```
git clone https://github.com/syuyuyu/portfolio.git
```

3.開啟終端機(Terminal)進入專案資料夾，輸入

```
npm install
```

4.終端機輸入開啟專案

```
npm start
```

5.瀏覽器開啟畫面成功
